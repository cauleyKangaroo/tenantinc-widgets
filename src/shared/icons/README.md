# Shared icons

Icons from the Figma "Pika Icons v2.2" set. Two variants per icon:

- `<name>-stroke.svg` — linework (outline)
- `<name>-solid.svg` — filled

All are 24×24 and drawn with `currentColor`, so they inherit the surrounding
text `color` (stroke icons via `stroke`, solid via `fill`). Drop them into a
component as inline SVG (copy the `<path>`s) or import the file as an asset.

## Available (core batch)

map-pin · phone · message · star · chevron-down · chevron-left · chevron-right ·
chevron-up · search · filter · tag · info-circle · question-circle · share ·
play-circle · external-link
