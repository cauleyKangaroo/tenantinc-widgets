Tenantinc Widget Dev Harness

Bundles are loaded via require.js exactly as Duda does — AMD modules at /dist/widget-*.js. Run npm run dev to start.

widget-hero

Changing the dropdown calls init() again with new props — the same thing Duda does when the site editor updates a content panel field.

widget-clock

widget-space-list

Dropdowns mirror the Duda content-panel vars (layoutMode, filterPosition, additionalPanelSection, additionalPanelPosition). The in-widget filter pills/checkboxes are React state. Note: an AP position that collides with the filter side falls back to bottom.
